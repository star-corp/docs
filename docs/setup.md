# Install

You can install netSys by cloning the repo.

## netSys Developer Build

Clone the dev branch

`git clone -b dev <URL>`

Edit the config file

`{
    "config": {
        "token": "token",
        "prefix": "-"
    
    }
    
}`

Run the startup command 

`node .` or `node main.js`
